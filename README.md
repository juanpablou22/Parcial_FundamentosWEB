# Parcial_FundamentosWEB
#  Diario Lácteo - Catálogo de Colección

Proyecto de práctica para el desarrollo de habilidades en *HTML* y *CSS, enfocado en diseño editorial, uso de **CSS Grid, **Flexbox* y *Responsive Design*.

##  Características
* *Diseño Editorial:* Estética moderna basada en tipografías pesadas y espacios limpios.
* *Layout Adaptable:* Galería de productos optimizada para dispositivos móviles y escritorio.
* *Footer Estilizado:* Basado en una estética minimalista con enlaces de gran formato.
* *Responsive:* Navegación y componentes totalmente funcionales en cualquier pantalla.

##  Tecnologías utilizadas
* *HTML5:* Estructura semántica.
* *CSS3:* Estilos personalizados, Grid, Flexbox y Media Queries.
